# EDUPLAN v8 corregida

Versión funcional que conserva la presentación visual y recupera la posibilidad de agregar, editar y eliminar hallazgos y acciones evaluables.

## Archivos principales

- `index.html`: versión editable local.
- `visualizador.html`: versión pública de solo visualización.
- `app.js`: lógica de EDUPLAN.
- `styles.css`: estilos.
- `logo_pef_blanco_ua.png`: logo institucional.

## Publicación en GitHub Pages

Sube todos los archivos internos de esta carpeta al repositorio y activa Pages en `main / root`.

Enlace editable local/publicado:
`https://TUUSUARIO.github.io/Eduplan/`

Enlace de solo visualización:
`https://TUUSUARIO.github.io/Eduplan/visualizador.html`

También puedes usar:
`https://TUUSUARIO.github.io/Eduplan/?readonly=1`

Importante: esta versión no tiene base de datos central. Los cambios se guardan en el navegador de quien edita. Para colaboración real multiusuario se requiere Supabase u otra base de datos.
