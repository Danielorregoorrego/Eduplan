# EDUPLAN v7

Sistema local editable para gestión, seguimiento, evaluación y reportabilidad de planes de acreditación de carreras de pedagogía.

## Mejoras v7
- Encabezado lateral corregido: logo y nombre EDUPLAN ya no se superponen.
- Descarga de reportes en TXT, HTML, Word compatible (.doc), Excel compatible (.xls).
- Resumen del dashboard descargable en TXT, Word compatible (.doc), Excel compatible (.xls).
- Botón PDF abre una vista imprimible para usar “Guardar como PDF” desde el navegador.

## Uso
Descomprimir el ZIP y abrir `index.html` en el navegador.

## Nota
Esta versión guarda los datos en el navegador mediante localStorage. Para trabajo multiusuario real se debe migrar a una versión web con base de datos central, autenticación y roles.
